# DECISIONS — Decisiones del MVP

## D001 — Sitio estático primero

Se usa Astro estático para velocidad, SEO y simplicidad. La lógica mínima de leads vive en un Worker.

## D002 — Pago externo

No se construye checkout propio. El MVP usa links externos de pago para salir rápido.

## D003 — Compra como CTA primario

WhatsApp no será el CTA principal. La conversación se usa para dudas y cierre manual, no para reemplazar la página de venta.

## D004 — No login, no admin

El contenido se edita desde archivos TypeScript. Esto reduce complejidad y permite publicar en horas.

## D005 — Aviso legal visible

Se incluye aviso de material independiente para reducir confusión comercial con Ceneval.

## D006 — Mobile-first real

La experiencia móvil manda. Escritorio se adapta después.
