# TASKS — Ejecución en horas

## Bloque 0 — Configuración inicial

- [ ] Instalar dependencias.
- [ ] Ejecutar `npm run dev`.
- [ ] Confirmar que `/` carga.
- [ ] Cambiar datos reales en `src/content/site.ts`.
- [ ] Cambiar precios y links en `src/content/product.ts`.

## Bloque 1 — Contenido mínimo vendible

- [ ] Colocar portada real o mockup en `public/images/`.
- [ ] Ajustar texto del hero.
- [ ] Definir diferencias reales entre Clásica y Plus.
- [ ] Definir precio de Clásica.
- [ ] Definir precio de Plus.
- [ ] Definir ciudades con entrega personal.
- [ ] Definir método de envío nacional.

## Bloque 2 — Conversión

- [ ] Configurar link de WhatsApp Business.
- [ ] Configurar Facebook real.
- [ ] Configurar links de pago externos.
- [ ] Revisar CTAs en móvil.
- [ ] Probar sticky CTA.

## Bloque 3 — Formulario

- [ ] Crear webhook externo.
- [ ] Configurar `LEADS_WEBHOOK_URL`.
- [ ] Probar `POST /api/leads`.
- [ ] Confirmar recepción del lead.
- [ ] Añadir Turnstile si hay spam.

## Bloque 4 — Publicación

- [ ] Ejecutar `npm run build`.
- [ ] Ejecutar `npm run worker:dev`.
- [ ] Ejecutar `npm run deploy`.
- [ ] Apuntar dominio/subdominio en Cloudflare.
- [ ] Probar desde celular con datos móviles.

## Bloque 5 — Revisión final

- [ ] Revisar ortografía.
- [ ] Revisar que no haya promesas absolutas de admisión.
- [ ] Revisar que EXANI-II® tenga aviso de independencia.
- [ ] Revisar que el botón comprar no mande a links vacíos.
- [ ] Revisar que WhatsApp tenga mensaje precargado.
